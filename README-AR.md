# 🎮 مسابقة العيد - Rahoot Arabic Edition

نسخة معربة من مشروع Rahoot مخصصة للمسابقات العائلية في العيد.

## 🚀 التشغيل السريع

### باستخدام Docker (الأسهل):

```bash
cd Rahoot-Arabic-Edition
docker compose up -d
```

**واجهة اللاعبين:** `http://localhost:3000`
**لوحة تحكم المدير (لإنشاء غرفة):** `http://localhost:3000/manager` (كلمة المرور: `123`)

### بدون Docker:

```bash
cd Rahoot-Arabic-Edition
pnpm install
pnpm run dev
```

## 📝 التعديلات المطلوبة لتعريب كامل

### 1. تعريب النصوص في الواجهة

**الملفات المطلوبة:**
```
packages/web/src/features/game/components/
├── join/Room.tsx          # نص "أدخل كود الغرفة"
├── join/Username.tsx      # نص "أدخل اسمك"
├── states/Question.tsx    # عرض السؤال
├── states/Answers.tsx     # الإجابات
└── states/Result.tsx      # النتائج
```

### 2. إضافة المزيد من الأسئلة

**في مجلد `config/quizz/`:**
- يمكنك إضافة ملفات JSON جديدة لكل قسم أو تصنيف ترغب فيه.
- تحتوي النسخة الحالية على 80 سؤالاً منقحاً مع ميزة الترتيب العشوائي التلقائي للإجابات لتجنب الحفظ.

### 3. تعديل Socket.io Server

**في `packages/socket/src/services/game.ts`:**
- تأكد من دعم RTL
- أضف رسائل خطأ بالعربية
